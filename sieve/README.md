# Sieve

Prime numbers generator

## Installation

Add this line to your application's Gemfile:

```ruby
#gem 'sieve'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sieve

## Usage

sieve [prime_index_limit]

E.g.:
$sieve 10

challenges/sieve$ sieve 10
